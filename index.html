TEX: Test-time Scaling Coding Agents via Execution-based Cross-Validation

Akash Gokul, Srijan Bansal, Hiroaki Hayashi, Semih Yavuz, Yingbo Zhou
Salesforce Research

Abstract

We introduce TEX, a novel approach for test-time scaling of testing and coding agents using execution-based cross-validation. Inference-time scaling of coding and testing agents has largely focused on two simple paradigms: parallel scaling (sampling N candidates in parallel) or sequential scaling (i.e. thinking for more tokens). In this work, we propose a scaling strategy that combines the benefits of parallel scaling, namely diversity, with sequential scaling. In particular, TEX runs multiple coding agents in parallelâ€” each generating a test and code solutions for a given problem. Then, we perform a cross-candidate execution where code solutions from each candidate are executed against test scripts from other candidates, and vice versa. We then let each agent continue scaling sequentially, but now additionally conditioned on how its test and code solutions fare against other candidates. Our results show that in real-world test-case generation benchmarks such as SWT-Bench, TEX significantly improves pass@1 (+7%). Further, using majority voting (i.e. running generated tests against generated code patches) leads to state-of-the-art performance (86.4%) on SWT-Bench when using Claude 4 Sonnet and sampling just 4 parallel candidates.
